# Incubyte QA Assignment

## Overview

This repository contains my submission for the Incubyte QA Assignment. The assignment focuses on testing the Gmail Compose functionality and includes both BDD and Traditional Test Cases.

## Repository Contents

### 1. BDD Test Cases

The BDD test cases have been structured with the following columns:

* Scenario ID
* Scenario
* Given
* When
* Then
* Type (Positive, Negative, Edge)
* Priority

### 2. Traditional Test Cases

The Traditional Test Cases sheet includes:

* Test Case ID
* Test Case Title
* Preconditions
* Test Steps
* Expected Result
* Actual Result
* Status
* Browser
